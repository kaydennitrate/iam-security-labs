Design and execution of granular access control policies to manage the user lifecycle and ensure network compliance.
Technical Specifications
Gateway: Huawei Security/ACL Engine
Standards: WPA3, 802.11ax
Management: Access Control Lists (ACL), QoS Throttling
Security Controls Implemented
Network Segmentation: Defined Access Control Lists (ACLs) to restrict inter-device communication and segment guest traffic.
Lifecycle Management: Established time-based access schedules and automated session termination for non-compliant identities.
Compliance Enforcement: Leveraged MAC filtering and bandwidth management to ensure resource availability for critical business functions.
